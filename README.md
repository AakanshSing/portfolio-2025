# Beautiful React Tailwind Portfolio

This is a beautiful portfolio template built with React and Tailwind CSS, based on the tutorial by PedroTech.

## Overview

This portfolio was created by following [PedroTech's detailed YouTube tutorial](https://youtu.be/ifOJ0R5UQOc?si=c4o4OHD7vFsFqd_8). The tutorial provides clear, step-by-step guidance making it easy to build your own portfolio.

## Customizations

The following sections have been customized from the original template:

- **Message Form** - Modified with custom styling and functionality
- **Projects** - Updated project showcase section
- **Skills** - Revised skills display

## Getting Started

To create a similar portfolio, follow along with [PedroTech's YouTube video](https://youtu.be/ifOJ0R5UQOc?si=c4o4OHD7vFsFqd_8). The tutorial covers all essential steps in great detail.
